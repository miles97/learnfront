快讯：李荣浩荣获今晚最佳男歌手

2014年，李荣浩凭借个人的首张专辑《模特》拿下第25届金曲奖最佳新人奖，这是内地歌手首次在金曲奖斩获奖项。同时因为参加《我是歌手》节目，让这个音乐才子在内地一夕爆红。实际上，今年30岁的李荣浩在音乐制作圈中已工作多年，帮赵薇、陈坤、杨丞琳、李宇春都创作过歌曲。《李白》、《模特》、《太坦白》…… 他的每首歌几乎都能单曲循环很久，节奏轻快，音色极具辨识度，唱尽了都市人的落寞与孤单。 李荣浩：我只做自己喜欢的事情 9岁时，家人送给他一把吉他做生日礼物，开始他和家人都没有把这当回事，只觉得吉他就是个玩具，结果李荣浩自弹自唱、自娱自乐，对吉他感情渐深。当时钢琴老师和吉他老师都找不到，就跟着磁带学，反复尝试找到“那个”音调。“那种笨方法让自己耳朵变得更灵敏，但那时不懂是在练习耳朵，我就知道去找那（音调），所以走了不少弯路。”李荣浩语气轻松，神情显然有些怀念刚开始玩吉他的年代。 对于吉他日渐痴迷的李荣浩，终于因为不顾学习，只顾吉他而遭到家长的极度反对，“他们把琴锁起来了，我就把自己锁起来了，就是谁也别进来，我就在自己的房间里。”后来赢得了父母妥协，不过不是学吉他，而是学低音提琴，就这样李荣浩开始了每周末从蚌埠坐火车到南京学琴的日子。但他并不喜欢低音提琴，觉得这只是一种妥协，至少这样，还能继续玩吉他了。这段在南京的学琴日子在他眼中是极其枯燥的，但得益于这段经历，他才有能力亲自编写专辑歌曲里所有的弦乐与交响乐谱。
